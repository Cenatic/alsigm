BÚSQUEDA RÁPIDA
Para realizar una Búsqueda Rápida de Elementos en Fondos Documentales se completarán uno o varios de los campos de la búsqueda:

• Ámbito: se seleccionará de la lista que aparece al pulsar la flecha de la derecha del campo. Si se quiere añadir más de un ámbito se pulsará el botón Añadir.

• Código: se introducirá manualmente el código del elemento a buscar.

• Título: se introducirá manualmente el título de elemento a buscar.

• Nivel: se seleccionará el nivel en el que se quiere realizar la búsqueda. Para seleccionar todos los niveles se pulsará en Seleccionar todos. Para quitar la selección realizada se pulsará en Quitar todos.

• Estado: se seleccionará el estado del elemento a buscar.

Para realizar una búsqueda es obligatorio por lo menos seleccionar el nivel de búsqueda. Hay que tener en cuenta, que cuantos más datos se introduzcan más concretos serán los resultados de la búsqueda realizada.

Una vez introducidos los datos correspondientes se pulsará Buscar.
