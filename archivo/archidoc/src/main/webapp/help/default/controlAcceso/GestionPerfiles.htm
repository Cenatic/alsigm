GESTIÓN DE PERFILES
En la pantalla de Gestión de perfiles se muestran los perfiles pertenecientes a la aplicación en un listado.

Se visualizarán los perfiles en función del módulo que se haya seleccionado. El módulo se selecciona de la lista desplegable que muestra el sistema al pulsar sobre la flecha de la derecha del campo Módulo.

Para consultar cualquiera de los perfiles que aparecen en pantalla, pulsar sobre el perfil a consultar.

Pulsar Crear para crear un nuevo perfil dentro de un módulo específico.

Para eliminar un usuario, se seleccionará de la lista y se pulsará Eliminar.

Se puede buscar un perfil en concreto utilizando la opción Buscar.

Para volver a la pantalla principal se pulsará Cerrar.
