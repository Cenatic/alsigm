GESTIONAR TAREA DE DIGITALIZACIÓN
Bandeja con las tareas pendientes y tareas a gestionar de digitalización que tiene el usuario conectado.

En las tareas pendientes se muesran aquellas tareas que se acaban de crear.

Se seleccionará la tarea con la que se quiere trabajar pulsando sobre el Título de la misma.

Se introducirán los documentos a digitalizar pulsando Añadir, se pulsará Finalizar captura y Aceptar para Validar la tarea de digitalización.

En Tareas a gestionar el usaurio podrá crear las tareas que quiere realizar pulsando Crear.
