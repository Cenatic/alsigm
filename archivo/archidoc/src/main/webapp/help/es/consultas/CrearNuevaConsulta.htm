CREAR NUEVA CONSULTA
Introducir los datos correspondientes para Crear Nueva Consulta en función del Tipo de Solicitante que se seleccione:

  • Seleccionando Ciudadano se completará el Nombre, el Tema, el Motivo y el Archivo

  • Seleccionando Órgano se completará el Órgano, el Representante, el tema, el Motivo y el Archivo

  • Seleccionando Investigador se completará el Usuario, el Tema, el Motivo y el Archivo

Una vez introducidos los datos, pulsar Aceptar para crear la nueva consulta.
