Ejecución de acciones multientidad:

Esta acción permite realizar operaciones predefinidas sobre varias entidades.

Dependiendo de cada acción es posible que se nos soliciten varios datos:

  • Entidades origen
  • Entidades destino
  • Fichero
  • Opciones de la acción
Finalmente se mostrará una pantalla resumen desde donde se podrá ejecutar la acción.
