PAGO
La pantalla de Pago electrónico se divide en tres opciones:
  • Listado de liquidaciones pendientes del ciudadano
  • Listado liquidaciones
  • Autoliquidaciones

  • Listado de liquidaciones pendientes:
    • Se muestran las liquidaciones pendientes del ciudadano conectado.En el listado se muestra el Periodo de Pago (rango de fechas en las que el ciudadano debe realizar el pago), Importe (importe que el ciudadano debe pagar) y Vencimiento (En plazo, Menos de un mes, Caducado)

    Para proceder al pago, pulsar en el botón Realizar pago de la derecha de la liquidación correspondiente.

    El sistema muestra un formulario de pago. Seleccionar la forma de realizar el pago, es decir, pago con tarjeta de crédito o a través de entidad bancaria.En función del tipo de pago seleccionado, completar los datos correspondientes y pulsar Enviar.

    Una vez enviado el pago, el sistema muestra un resumen para verificar los datos por el ciudadano. Una vez verificados, pulsar Firmar y Enviar.

    El sistema muestra las firmas correspondientes al ciudadano. Seleccionar la firma correspondiente.

    Una vez seleccionado el certificado, el sistema muestra el justificante de pago, con un Número de Referencia (dado por el organismo público) y un Número de Referencia Completo (incluye el número de referencia del organismo público y el número de referencia de la entidad bancaria referido al pago realizado)

    Desde esta pantalla, se puede imprimir el justificante de pago mediante el botón Justificante.

    Para volver a la pantalla de liquidaciones pendientes, pulsar Volver.

  • Listado Liquidaciones:
    • Se muestran las liquidaciones, pendientes o no, del ciudadano y permite realizar una búsqueda.

    Para realizar una búsqueda, completar los datos correspondientes (Referencia o Referencia completa) y pulsar Buscar. El resultado se muestra en formato lista.

    En la lista de resultados se podrán consultar las liquidaciones que estén pagadas y realizar pago en aquellas que estén pendientes de pago.

  • Autoliquidación:
    • Se muestran las liquidaciones solicitadas por el ciudadano.

    Para proceder a un pago de una Autoliquidación, pulsar el botón Enviar.

    En la parte inferior de la pantalla se muestra un Histórico de las liquidaciones correspondientes al ciudadano.

    En esta pantalla, se puede realizar una búsqueda de liquidaciones tanto por Referencia como por Referencia completa. El resultado se muestra en formato lista.
