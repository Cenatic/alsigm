Componentes >> Reglas Atrás

En el catálogo de Reglas se muestran todas las reglas dadas de alta en el sistema.

Una regla es una clase java que implementa el interfaz IRule, definido en la documentación del sistema.

Para dar de alta una regla, anteriormente hay que implementar su clase java de acuerdo a la estructura definida en la documentación, y almacenarla en una ruta accesible desde el servidor de aplicaciones.

Una vez realizado esto, desde el catálogo de reglas, pulsar Nueva regla, que mostrará una pantalla donde se solicitará la siguiente información:

  • Nombre:
    • nombre de la regla. No puede tener espacios en blanco, ya que será invocada desde los tag de documento.

  • Descripción: indicaciones de la funcionalidad.

  • Clase: indica la ruta donde se ha dejado la clase en el servidor.


Se debe consultar el manual de usuario del Catálogo de Procedimientos para obtener la definición del interfaz IRule.
