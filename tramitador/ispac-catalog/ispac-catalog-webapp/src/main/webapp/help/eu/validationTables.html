Componentes >> Tablas de Validación Atrás

En el catálogo de Tablas de Validación se muestran todas las tablas de validación dadas de alta en el sistema. Las tablas de validación se usarán en las entidades, para garantizar que los valores de un determinado campo serán elegidos entre unos determinados, definidos en dicha tabla de validación.

Al crear una tabla de validación, se creará en base da datos la tabla física correspondiente.


Hay disponibles dos tipos de tablas de validación: simples y con sustituto:

Las tablas de validación simples son listas de valores posibles para un campo.

Las tablas de validación con sustituto recogen pares valor-sustituto; el campo valor será almacenado en base de datos, mientras que en los formularios aparecerá el sustituto correspondiente,para un campo con este tipo de validación.

Al dar de alta una tabla de validación se pedirá la siguiente información:

  • Tabla: nombre de la tabla en base de datos


  • Nombre: nombre en la aplicación para la tabla de validación . Para crear uno nuevo pulsar la opción “Añadir”.

  • Tipo: simple o con sustituto.

  • Tamaño campo VALOR: tamaño de dicho campo, por defecto será un texto de 10 caractéres.

  • Tamaño campo SUSTITUTO: tamaño de dicho campo, en las tablas con sustituto.

Una vez creada una tabla, se puede visualizar su información, y añadir, modificar o eliminar valores a la misma, así como marcar un valor como vigente. Un valor no vigente no se visualizará entre los disponibles para la validación de un campo.
